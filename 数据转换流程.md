# 数据转换流程：GVHMR → GMR → RL训练

## 完整流程图

```
视频
  ↓
GVHMR (提取3D人体姿态)
  ↓
gvhmr_to_robot.py (GMR脚本)
  ↓
pkl文件 (机器人关节轨迹)
  ↓
batch_gmr_pkl_to_csv.py
  ↓
CSV文件 (36列: 3+4+29)
  ↓
csv_to_npz.py (IsaacLab计算导数)
  ↓
NPZ文件 (12个数组)
  ↓
train.py (RL训练)
```

## 各阶段数据格式

### 1. GVHMR输出 → GMR输入
**格式**: SMPL-X 3D关节位置
**脚本**: `gvhmr_to_robot.py` 
**输出**: `pkl` 文件 (Python dict)

### 2. pickle (pkl) 格式
```python
{
    'fps': 30,                      # 帧率
    'root_pos': (N, 3),             # 根部位置 [x, y, z]
    'root_rot': (N, 4),             # 根部旋转 [x, y, z, w]
    'dof_pos': (N, 29)              # 29个关节角 [rad]
}
```

### 3. CSV 格式
**36列矩阵**: 
```
root_pos[3] | root_rot[4] | dof_pos[29]
x,y,z       | qx,qy,qz,qw| joint_1...29
```
**脚本**: `batch_gmr_pkl_to_csv.py`

### 4. NPZ 格式 (IsaacLab计算)
**12个数组**:
- `root_pos`: 位置 (N, 3)
- `root_rot`: 旋转 (N, 4)
- `root_vel`: 线速度 (N, 3) ← FK导数
- `root_ang_vel`: 角速度 (N, 3) ← FK导数
- `dof_pos`: 关节角 (N, 29)
- `dof_vel`: 关节速度 (N, 29) ← FK导数
- `dof_acc`: 关节加速度 (N, 29) ← FK导数

**脚本**: `csv_to_npz.py` (需IsaacLab)

### 例2: pkl转CSV (使用GMR官方脚本) ⭐
GMR作者提供了 `batch_gmr_pkl_to_csv.py` 脚本，可以批量转换所有pkl文件到CSV格式。

**单个文件转换**:
```bash
cd /home/abc/GMR

# 激活GMR环境
source /home/abc/miniconda3/bin/activate gmr

# 转换指定pkl文件（一次性）
python -c "
import pickle
import numpy as np
import os

pkl_file = 'pkl_files/walk_amass.pkl'
with open(pkl_file, 'rb') as f:
    motion_data = pickle.load(f)

# 提取数据: root_pos(3) + root_rot(4) + dof_pos(29) = 36列
dof_pos = motion_data['dof_pos']
motion = np.zeros((dof_pos.shape[0], dof_pos.shape[1] + 7), dtype=np.float32)
motion[:, :3] = motion_data['root_pos']
motion[:, 3:7] = motion_data['root_rot']
motion[:, 7:] = dof_pos

# 下采样处理（如果fps > 30）
frame_rate = motion_data['fps']
if frame_rate > 30:
    downsample_factor = frame_rate / 30.0
    indices = np.arange(0, motion.shape[0], downsample_factor).astype(int)
    motion = motion[indices]
    print(f'Down-sampled from {dof_pos.shape[0]} to {motion.shape[0]} frames')

# 保存CSV
os.makedirs('pkl_files/csv', exist_ok=True)
np.savetxt('pkl_files/csv/walk_amass.csv', motion, delimiter=',')
print(f'Saved: {motion.shape[0]} frames x {motion.shape[1]} columns')
"
```

**批量转换所有pkl文件** (推荐):
```bash
cd /home/abc/GMR
source /home/abc/miniconda3/bin/activate gmr

# 使用GMR官方脚本批量转换
python scripts/batch_gmr_pkl_to_csv.py --folder pkl_files

# 或指定其他pkl文件夹
python scripts/batch_gmr_pkl_to_csv.py --folder /home/abc/whole_body_tracking/pklfiles
```

**脚本说明**:
- 输入: `--folder` 指定包含pkl文件的目录
- 输出: 在该目录下自动创建 `csv/` 子目录，生成同名的CSV文件
- 自动处理: fps>30时自动下采样到30fps
- 格式: 36列 = root_pos(3) + root_rot(4) + dof_pos(29)

## 关键点总结

| 步骤 | 脚本 | 输入 | 输出 | 耗时 | 环境 |
|------|------|------|------|------|------|
| GVHMR | - | 视频 | SMPL-X关节 | 1-5分钟 | GVHMR |
| 人体→机器人 | `gvhmr_to_robot.py` | SMPL-X | pkl (关节角) | 几秒 | gmr |
| pkl→CSV | `batch_gmr_pkl_to_csv.py` | pkl | CSV (36列) | 秒级 | gmr |
| **CSV→NPZ** | **csv_to_npz.py** | CSV | NPZ (+速度+加速度) | **5-10分钟** | **env_isaaclab** |
| 训练 | `train.py` | NPZ | RL模型 | 1-2天 | env_isaaclab |
