# GMR → IsaacLab训练快速开始

## 🚀 5分钟快速开始

### 前置条件
- ✅ GMR已生成CSV文件 (`/home/abc/GMR/pkl_files/csv/`)
- ✅ IsaacLab环境已安装
- ✅ whole_body_tracking已安装

### Step 1: 登录WandB并创建Registry

```bash
# 1. 登录WandB
wandb login
# 输入API key (从 https://wandb.ai/authorize 获取)

# 2. 访问 https://wandb.ai/ 创建Registry集合
# Core → Registry → New Collection
# Name: Motions, Type: All Types
```

### Step 2: 批量转换CSV到NPZ并上传

```bash
cd /home/abc/GMR

# 替换your-username为你的WandB用户名
python programs/batch_csv_to_npz.py \
    --wandb-org your-username \
    --headless

# 示例: 
# python programs/batch_csv_to_npz.py --wandb-org ruixiao756-lgtm --headless
```

**预计时间**: 每个动作1-2分钟

### Step 3: 验证上传

```bash
# 检查Registry中的动作
python programs/check_wandb_registry.py --wandb-org your-username

# 回放测试(验证上传成功)
cd /home/abc/whole_body_tracking
python scripts/replay_npz.py \
    --registry_name=your-username-org/wandb-registry-motions/stand_g1
```

### Step 4: 开始训练

```bash
cd /home/abc/GMR

# 训练所有动作
python programs/batch_train_motions.py --wandb-org your-username

# 或只训练特定动作
python programs/batch_train_motions.py \
    --wandb-org your-username \
    --motions stand_g1 walk_amass turn_amass
```

**预计时间**: 
- 简单动作(stand): ~1小时
- 中等动作(walk/turn): ~2小时  
- 复杂动作(dance): ~5小时

### Step 5: 查看训练结果

访问: `https://wandb.ai/your-username/GMR-MotionTracking`

## 🛠️ 脚本说明

### 批量转换脚本
```bash
programs/batch_csv_to_npz.py
```
将GMR的CSV文件转换为NPZ并上传到WandB Registry

**常用参数**:
- `--wandb-org`: 你的WandB用户名 **(必需)**
- `--headless`: 无头模式,不显示UI
- `--filter`: 过滤文件,如`--filter "walk*"`只处理行走动作
- `--dry-run`: 测试模式,不实际执行

**示例**:
```bash
# 转换所有动作
python programs/batch_csv_to_npz.py --wandb-org ruixiao756-lgtm --headless

# 只转换walk相关动作
python programs/batch_csv_to_npz.py --wandb-org ruixiao756-lgtm --filter "walk*" --headless

# 测试(不实际执行)
python programs/batch_csv_to_npz.py --wandb-org ruixiao756-lgtm --dry-run
```

### 批量训练脚本
```bash
programs/batch_train_motions.py
```
批量训练多个动作

**常用参数**:
- `--wandb-org`: 你的WandB用户名 **(必需)**
- `--motions`: 要训练的动作列表(默认全部)
- `--max-iterations`: 训练迭代次数(覆盖默认配置)
- `--num-envs`: 并行环境数(默认4096)
- `--continue-on-failure`: 某个动作失败后继续训练

**示例**:
```bash
# 训练所有动作(使用默认配置)
python programs/batch_train_motions.py --wandb-org ruixiao756-lgtm

# 训练特定动作
python programs/batch_train_motions.py \
    --wandb-org ruixiao756-lgtm \
    --motions stand_g1 walk_amass

# 自定义训练参数
python programs/batch_train_motions.py \
    --wandb-org ruixiao756-lgtm \
    --motions walk_amass \
    --max-iterations 2000 \
    --num-envs 8192

# 失败后继续训练
python programs/batch_train_motions.py \
    --wandb-org ruixiao756-lgtm \
    --continue-on-failure
```

### Registry检查脚本
```bash
programs/check_wandb_registry.py
```
查看WandB Registry中已上传的动作

**示例**:
```bash
python programs/check_wandb_registry.py --wandb-org ruixiao756-lgtm
```

## ⚙️ 自定义配置

### 修改训练参数

编辑 `programs/batch_train_motions.py` 中的 `MOTION_CONFIGS`:

```python
MOTION_CONFIGS = {
    "walk_amass": {
        "num_envs": 4096,      # 并行环境数
        "max_iterations": 1000, # 训练迭代次数
        "description": "行走动作"
    },
    # 添加新动作配置...
}
```

### 添加新动作

1. GMR生成新的pkl和csv文件
2. 运行转换: `python programs/batch_csv_to_npz.py ...`
3. 训练新动作: `python programs/batch_train_motions.py --motions new_motion_name ...`

## 🔍 故障排查

### 问题: 找不到CSV文件
```bash
# 检查CSV文件
ls /home/abc/GMR/pkl_files/csv/

# 如果没有CSV文件,需要先从pkl生成
# (GMR应该已经生成了)
```

### 问题: WandB上传失败
```bash
# 检查登录状态
wandb status

# 重新登录
wandb login --relogin

# 测试网络连接
python -c "import wandb; api = wandb.Api(); print('OK')"
```

### 问题: 训练GPU内存不足
```bash
# 减少并行环境数
python programs/batch_train_motions.py \
    --wandb-org your-username \
    --num-envs 2048  # 从4096降到2048
```

### 问题: IsaacLab找不到Registry
```bash
# 检查Registry路径格式
# 正确: username-org/wandb-registry-motions/motion_name:latest
# 错误: username/wandb-registry-motions/motion_name

# 验证Registry中的动作
python programs/check_wandb_registry.py --wandb-org your-username
```
